## 如何创建新文章

```bash
hexo new "文章标题"
```

然后编辑 `source/_posts/` 目录下的 markdown 文件。

## 发布流程

```bash
# 本地预览
hexo server

# 生成静态文件并部署
hexo clean && hexo generate && hexo deploy
```
